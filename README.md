# Word Guessing Game

A command-line word guessing game built in Python. The player has 6 attempts
to guess a hidden word by suggesting one letter at a time.

## Features

- Random word selection from a built-in word list
- Input validation (handles duplicates, numbers, symbols)
- Tracks guessed letters and remaining attempts
- Play again without restarting the program

## How to run

Make sure you have Python 3 installed, then:
```bash
git clone https://github.com/rawanmismail/word-guessing-game.git
cd word-guessing-game
python game.py
```
